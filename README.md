# 📊 github-user-activity

> **Portfolio Project** — A simple API calling to summarize recent core user activity on GitHub.

![Status](https://img.shields.io/badge/status-complete-green)

---

## 📌 About

**github-user-activity** is a simple project from [roadmap.sh](https://roadmap.sh/projects/github-user-activity) to handle external API calls, format the response and properly return it to the user.

---

## 🏗️ Project Structure

```
github-user-activity/
├── src/
│   ├── api.ts/               # external API call
│   ├── formatter.ts/         # format output
│   ├── index.ts/
│   ├── types.ts/
├── .prettierrc
├── eslint.config.js
├── package.json
├── README.md
└── tsconfig.json
```

---

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/barbosacaio/github-user-activity.git
   cd github-user-activity
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Execute the program**

   ```bash
   npm run dev <username>
   ```
